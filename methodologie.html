🌀 Le cycle de Kolb

Ludus Mentis utilise le cycle d’apprentissage expérientiel de Kolb pour structurer ses ateliers :

  • Expérience concrète (jeu coopératif en équipe)
  • Observation et réflexion (débriefing)
  • Conceptualisation (mise en lien avec les soft skills)
  • Expérimentation active (nouveaux essais / défis)

🎯 Le modèle SAFE (CASEL)

Nos modules suivent les principes du modèle SAFE reconnu par le CASEL pour le développement des compétences socio-émotionnelles :

  • Séquenté : progression structurée dans les apprentissages
  • Actif : implication réelle des jeunes
  • Focalisé : chaque module cible une compétence précise
  • Explicite : liens clairs entre l’activité et les compétences développées
